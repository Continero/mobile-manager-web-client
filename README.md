# mobile-manager-web-free

## How to compile and run the website
 1. Install NodeJs
 2. npm install -g less
 3. npm install -g less-watch-compiler
 4. npm install
 5. npm run build-css
 6. start less-watch-compiler
 7. npm run start

 ## Running tests
 npm run test

 ## Debugging tests
 1. Use VS Code
 2. Press F5
